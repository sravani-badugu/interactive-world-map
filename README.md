# GeoPop: Interactive World Map 🌍

GeoPop is a professional, responsive, and interactive world map visualization tool. It provides an intuitive interface to explore global population and density data using an interactive map, search functionality, and detailed country insights.

## Features ✨
- **Interactive Map:** Powered by Leaflet.js, featuring smooth zoom and pan controls.
- **Data Visualization:** Choropleth map overlay displaying population density variations.
- **Global Search:** Quickly find any country with real-time search suggestions.
- **Detailed Info Panel:** View specific statistics like Population, Area, Capital, and Density for any selected country.
- **Country Comparison:** Compare statistics between two countries directly from the side panel.
- **Dark/Light Mode:** Seamless theme toggling for an optimal viewing experience.
- **Glassmorphism UI:** Modern, premium aesthetic with smooth transitions and responsive design.

## Technologies Used 🛠️
- HTML5
- CSS3 (Custom variables, Flexbox/Grid, Glassmorphism)
- Vanilla JavaScript
- Leaflet.js (Map rendering)
- FontAwesome (Icons)

## Setup Instructions 🚀
1. Clone the repository.
2. Open `index.html` in your web browser or use a local server like Live Server in VS Code.
3. No build steps or package installations are required!
